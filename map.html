Sound of Birth and Death

This is an interactive web interface where a user can visualize and hear the births and deaths occuring in real time across the globe. Teal denotes birth and maroon denotes death. The user can select upto six different musical instruments simultaneously to play music. Birth denotes a major note and death denotes a minor note. The value of each note is randomly selected from a pool of pre-selected notes. For each country, birth and death rate is defined in terms of beats per minute (BPM).
